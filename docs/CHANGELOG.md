## [0.1.0] — 2026-03-15

### Добавлено — Unity (игра)

**Движение игрока**
- CharacterController — ходьба, бег, прыжок, приседание
- Coyote time (0.15s) для прыжков с края платформы
- FOV kick при беге (normal → sprint FOV через Lerp)
- Head-bob через Cinemachine Noise
- Landing squash эффект при приземлении

**Система здоровья и брони**
- `HealthComponent` с интерфейсом `IDamageable`
- Броня поглощает 70% урона первой, остаток идёт в HP
- UI шкалы HP и брони (заглушки)
- Hit-flash эффект экрана при получении урона

**Взрывные бочки**
- `ExplosiveBarrel` с radius damage falloff
- Урон убывает по расстоянию от центра взрыва
- Camera shake через `CinemachineImpulseSource`
- VFX: огонь, дым, искры (placeholder)
- Разлёт осколков через Physics

**Смерть и рестарт**
- Game Over экран
- Respawn из последнего checkpoint
- Сброс состояния игрока (HP, броня, позиция)
- `GameManager` с `GameState` enum (Playing, Paused, Dead)

**Архитектура**
- Компонентная архитектура — каждая система отдельный компонент
- Слои: Player, Enemy, Projectile, Ground, Interactable
- `GameEvent` ScriptableObject для развязки систем
- Базовая структура папок `Assets/_Project/`

### Известные проблемы
- UI шкалы HP/брони — только заглушки, без дизайна
- Визуал игрока — placeholder капсула
- Нет оружия — нельзя атаковать
- Нет врагов — тест только на бочках
- Уровень — серый ProBuilder тест без освещения
